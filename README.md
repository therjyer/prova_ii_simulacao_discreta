# prova_ii_simulacao_discreta
A Prova II de Simulação Discreta trata da simulação de um aeroporto durante o período de um dia de operações.  O trabalho deve ser feito para ser simulado computacionalmente, encontrando gargalos e propondo soluções que visem a melhorar os gargalos identificados. Os alunos devem demostrar senso crítico nas análises e soluções sugeridas.
